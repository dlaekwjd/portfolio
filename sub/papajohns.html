01Over View

소비자들을 끌어당길 수 있는 화장품 페이지 제작

화장품 브랜드 페이지 제작을 하였습니다. 그 안에서도 두가지 느낌으로 나눠 모든 사람들이 만족할 수 있는 느낌을 주고 싶었습니다.

02Detail View

크로스 부라우징과 웹 표준, 접근성을 만족하는 사이트 마크업과 jQuery를 이용해서 부족한 접근성 제어
각 a태그 href에 링크를 걸어 관련된 페이지로 이동할 수 있게 하였습니다.
이미지 alt 속성에 적절한 텍스트를 적어주었습니다.
스크롤을 내리면 일정시간에 도달했을때 서서히 글씨가 나타나도록 jQuery로 fade 효과를 주었습니다.
자동으로 실행되는 슬라이더에 인디케이터 버튼, 시작과 정지 버튼, 이전과 다음 버튼 모두 넣어 제어해주었습니다.
찍혀진 도장갯수와 해당갯수를 채워 쿠폰이 발급되었을때 사용자가 확인할 수 있도록 위에 표시해주었습니다.
